# Examples

It's gonna be a game!

# Setup

```
git clone https://github.com/mirrorcoloured/colyseus-example
cd colyseus-example
npm install
```

# Run
To run main server
`npm run start`

To run webpack bundler
`npm run build` or `npm run watch`

Open [http://localhost:2567](http://localhost:2567) in your browser.
